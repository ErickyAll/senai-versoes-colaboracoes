# senai-versoes-colaboracoes
Senai Full Stack

Senai Atividade de Versionamento

git init : Cria um novo projeto de git;

git config --global -e : Responsável pelas opções de configuração do GIT;

git status : Exibe status das alterações;

git clone : Cria uma cópia exata de um repositório já existente;

git remote : Conexão entre seu repositório local e um repositório remoto;

git push : serve para subir suas modificações para um repositório remoto conectado anteriormente com git remote;
